**question 1**
如何 更新 刷臉 ID ?
**answer1**
(1) 持 玉山 銀行 任一 張 有效 之 金融卡 到 玉山 銀行 營業部 、 城中 分行廳 內 之 刷臉 ATM 進行 更新 。
( 2) 在 ATM 螢幕 點選 「 無 卡 交易 」 → 「 刷臉 提款 」 → 「 更新 刷臉 ID 」 → 插入 「 金融卡 」 → 輸入 「 晶片 金融卡 密碼 」 → 系統 會 傳送 一 組 簡訊 密碼 至 您 的 手機 ， 請 您 查收 簡訊 並 於 ATM 輸入 「 簡訊 密碼 」 → 設定 「 刷臉 ID 」 即可 完成 變更 。
