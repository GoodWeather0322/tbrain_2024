**question 1**
如何 決定 海外 ETF/ 股票 的 委託 價格 ?
**answer1**
透過 本行 特定 金錢 信託 投資 海外 ETF/ 股票 可 採 「 限價 交易 ( Limit Order ) 」 及 「 市價 交易 ( Market Order ) 」 二 種 ， 惟 「 市價 交易 」 僅 適用 於 美國 市場 賣出 委託 交易 。


**question 2**
市價 交易 適用 於 哪 個 市場 ?
**answer2**
美國 市場


**question 3**
限價 交易 和 市價 交易 有 什麼 不同 ?
**answer 3**
「 限價 交易 ( Limit Order ) 」 允許 設定 特定 價格 進行 交易 ， 而 「 市價 交易 ( Market Order ) 」 則 是 以 當前 市場 價格 進行 交易 ， 但 市價 交易 僅 適用 於 美國 市場 賣出 委託 交易 。
