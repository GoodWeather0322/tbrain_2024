**question 1**
什麼 是 QR Code 轉帳 ？
**answer1**
QR Code 轉帳 可 透過 玉山 行動 銀行 內 QR Code 掃描器 ， 掃描 本行 與 支援 臺灣 Pay 合作 銀行 之 QR Code 轉帳 收款 碼 ， 取得 收款 者 的 銀行 代碼 、 銀行 帳號 、 轉帳 金額 ( 限有 設定 金額 的 QR Code ) ， 並 進行 轉帳 。


**question 2**
透過 什麼 方式 可以 進行 QR Code 轉帳 ？
**answer2**
玉山 行動 銀行 內 QR Code 掃描器


**question 3**
QRCode 轉帳 可以 取得 哪些 資訊 ？
**answer 3**
收款 者 的 銀行 代碼 、 銀行 帳號 、 轉帳 金額


**question 4**
QRCode 轉帳 適用 於 哪些 銀行 的 收款碼 ？
**answer 4**
本行 與 支援 臺灣 Pay 合作 銀行 之 QR Code 轉帳 收款 碼
